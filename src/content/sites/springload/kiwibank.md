Kiwibank is 100% New Zealand owned and is found in PostShops nationwide.
The site is rendered statically and hosted on AWS for fast delivery – you can't get much faster than serving the HTML file directly. Security is enhanced by default as the backend isn’t publicly accessible. It additionally provides a safe environment for the content management team to make changes before sign off and hitting publish.
