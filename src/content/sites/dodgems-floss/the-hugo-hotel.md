The Hugo, a boutique hotel situated right in the heart of Canterbury’s historic city centre.

**Data Driven**

We set out to build a data-driven website powered by a headless Wagtail CMS, enabling flexible and efficient content management. By decoupling the front end from the CMS, we ensured a modern and scalable architecture capable of delivering fast, dynamic, and interactive user experiences.

**React frontend**

For the front-end development, we chose Next.js, a robust React framework that supports server-side rendering and static generation, allowing for optimal performance, improved SEO, and seamless integration with headless Wagtail. Paired with Tailwind CSS for styling, enabling rapid UI development, consistent design, and streamlined customisation across components.
