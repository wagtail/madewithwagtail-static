Statclass (Статкласс in Russian) is a software and methodological solution for teaching statistics and other data analysis-related courses in universities, as well for implementation academic research. It comprises number of components: a world-famous IBM SPSS Statistics software, some new functions added by us, the Predictive Solutions company, and a large bank of test questions along with a test engine to organize student testing. Teachers might use the built-in tests or create their own.
