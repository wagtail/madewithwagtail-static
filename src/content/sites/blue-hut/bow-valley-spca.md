The Bow Valley SPCA provides the local community with an adoption centre following a no kill, no cage philosophy to shelter, care for, and rehome stray and abandoned dogs and cats.

Site includes integration with PetPoint animal database and auto posts new and recently adopted animals to social media. The site code is also available on GitHub so that other SPCAs can use the open source code to create their own animal rescue websites.
