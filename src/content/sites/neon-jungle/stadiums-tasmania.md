The Stadiums Tasmania website serves as a home for the corporate governance and operational activities for all of the stadiums in Tasmania.

The website also houses several microsites – one for each stadium – and utilises one central Wagtail CMS instance to manage each site.

This flexible set-up allows us to make changes efficiently across all sites, and makes content management a breeze on the client side.
