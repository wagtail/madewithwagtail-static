Working with designer Karma Creative, we built this Wagtail based site for the renowned Lewin Clinic Physiotherapists.

The project was a simple vanilla wagtail and template build, utilising the Wagtailmenus package for menu management.

Other technologies include Sengrid and Mailchimp.

The site runs on Google Cloud platform, utilising Google Cloud Storage, Google Cloud SQL and Cloud Run with the site packaged as a Docker Container.
