The BigPicnic project brings together the public, scientists, policy-makers and industry to help tackle the global challenge of food security. Their aims are to engender public engagement with the issues of food security, to generate debate leading to practical solutions.

Using research evidence to support it the site provides information on the issues and the wider context which can vary for different communities and populations globally. Along with details of events run around the world and a project updates section the site provides resources and publications relevant to food security as well as a 'Get involved' section targeted towards specified user groups.
